Getting Started

Create Your First Project

  1. 1 Open the htdocs folder (C:\devamp\htdocs)
  2. 2 Create a new folder for your project
  3. 3 Add your PHP files (start with index.php)
  4. 4 Access via http://localhost/yourproject
